Detailed Description

SL_STATS_RECORD - This structure contains the statistics maintained by the SPI library. By design each statistic field increments when the associated condition occurs. Each field will be initialized to zero by the C run-time library and is therefore NOT set to zero by SL_Initialize (by design) so that each field maintains the cumulative value.
