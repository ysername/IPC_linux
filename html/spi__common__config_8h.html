Detailed Description

SPI Library Common Configuration - spi_common_config.h This file contains common definitions required by the SPI communications software residing on both the the VP and GP. It is generated from the template file "spi_common_config_h.txt" that is supplied with each release of the SPI communications library.

NOTE: DO NOT CHANGE THE NAMES OF THE PARAMETERS IN THIS FILE, ONLY PARAMETER VALUES MAY BE CHANGED.
