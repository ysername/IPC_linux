95 /*
96 ** PS_RX_MSG_HANDLER_LIST - Populate this macro with a list of product specific
97 ** message handlers. Each message handler must be a function with the following signature:
98 **
99 ** void MsgHandlerFunction(int msg_size_bytes, U8 * const msg_data)
100 **
101 ** The list defines the set of "callback" functions that are invoked to process incoming
102 ** "regular" (not "status") messages. Callback functions are invoked when the SL_DispatchMsg
103 ** function is called following successful unloading of a regular message via the
104 ** SL_UnloadPacketMsg function.
105 **
106 ** The following is an example of how to format this list (note that the "\" character
107 ** MUST be the last character on the line). It is recommended that handler function names
108 ** be descriptive to increase usability.
109 **
110 ** void MsgId1Handler(int msg_size_bytes, U8 * const msg_data);
111 ** void MsgId2Handler(int msg_size_bytes, U8 * const msg_data);
112 ** void MsgId3Handler(int msg_size_bytes, U8 * const msg_data);
113 **
114 ** #define PS_RX_MSG_HANDLER_LIST \
115 ** MsgId1Handler, \
116 ** MsgId2Handler, \
117 ** MsgId3Handler,
118 */
