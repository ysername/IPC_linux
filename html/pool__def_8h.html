Detailed Description

Header file with datapool item IDs, control table, and data allocation. To add a new datapool item, do the following:

  1. Add an ID definition to the datapool ID enumeration, DP_ELEMENT_IDS.
  2. Add a storage entry for the datapool item to DP_ITEM_STORAGE_T.
  3. Add a datapool access table entry of type DP_ITEM_ENTRY_T for the datapool item.
